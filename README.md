# Features
- Add and remove displays
- Expand and edit displays
- Reorder and swap displays

# Usage
## Add display
<img width="960" height="540" alt="image" src="https://github.com/user-attachments/assets/18ee585a-de49-4aef-b218-1985665a8071" />

1. Press the plus button.
2. Enter the RTSP settings (Display Title, IP, Port, Path, FPS).
3. Press the OK button.
4. The RTSP stream will start shortly.

Up to 6 displays can be shown per page.

## Remove display
<img width="960" height="540" alt="image" src="https://github.com/user-attachments/assets/fdd90ae8-b2c1-4559-96cb-395b0a83d1e1" />

1. Press the X button on the top-right of the display.

## Expand and edit
<img width="960" height="540" alt="image" src="https://github.com/user-attachments/assets/eb300e3c-b8f3-4c71-9ce5-d89212be766a" />

1. Double-click the display.
2. Edit the display settings.
3. Press the Apply or OK button.

## Reorder displays
<img width="960" height="540" alt="image" src="https://github.com/user-attachments/assets/8be219bd-07bb-47a4-a5de-c99165212145" />

1. Drag the title of a display.
2. Drop it onto the display you want to swap positions with.

## Etc
- Press Esc to exit the program.
- Popups can be closed by pressing Esc.
