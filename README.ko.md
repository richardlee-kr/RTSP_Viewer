Unity와 GStreamer로 제작된 간단한 RTSP 스트림 뷰어입니다.

## 기능
- 디스플레이 추가 및 삭제
- 디스플레이 확대 및 설정 편집
- 디스플레이 순서 변경 및 위치 교환

## 사용 방법

### 디스플레이 추가
<img width="960" height="540" alt="image" src="https://github.com/user-attachments/assets/18ee585a-de49-4aef-b218-1985665a8071" />

1. 플러스(+) 버튼을 누릅니다.
2. RTSP 설정을 입력합니다. (디스플레이 제목, IP, 포트, 경로, FPS)
3. OK 버튼을 누릅니다.
4. 잠시 후 RTSP 스트림이 시작됩니다.

한 페이지당 최대 6개의 디스플레이를 표시할 수 있습니다.

### 디스플레이 삭제
<img width="960" height="540" alt="image" src="https://github.com/user-attachments/assets/fdd90ae8-b2c1-4559-96cb-395b0a83d1e1" />

1. 디스플레이 **우측 상단의 X 버튼**을 누릅니다.

### 확대 및 편집
<img width="960" height="540" alt="image" src="https://github.com/user-attachments/assets/eb300e3c-b8f3-4c71-9ce5-d89212be766a" />

1. 디스플레이를 더블 클릭합니다.
2. 디스플레이 설정을 수정합니다.
3. Apply 또는 OK 버튼을 누릅니다.

### 디스플레이 순서 변경
<img width="960" height="540" alt="image" src="https://github.com/user-attachments/assets/8be219bd-07bb-47a4-a5de-c99165212145" />

1. 디스플레이의 제목 영역을 드래그합니다.
2. 위치를 변경하고 싶은 디스플레이 위에 드롭합니다.

### 기타
- Esc 키를 누르면 프로그램이 종료됩니다.
- 팝업 창은 Esc 키로 닫을 수 있습니다.
